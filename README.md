# Coffee Sales Intelligence Dashboard

An interactive Power BI dashboard developed to analyze coffee shop sales performance and provide a clear view of revenue trends, transaction activity, and product performance.

## Overview

This project focuses on transforming raw sales data into meaningful business insights through effective data modeling, DAX calculations, and interactive visualizations.

## Key Features

* Sales Performance Analysis
* Transaction Monitoring
* Average Price Tracking
* Monthly Sales Trends
* Sales Distribution by Time of Day
* Product Performance Analysis
* KPI-Driven Reporting

## Technologies

* Power BI
* Power Query
* DAX
* Data Modeling

## Business Value

The dashboard enables stakeholders to monitor performance, identify sales patterns, and make informed decisions based on data.
